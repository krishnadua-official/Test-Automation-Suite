WiFi Router Test Automation Suite
A comprehensive Python-based test automation framework for validating WiFi router performance, connectivity, and stability using black-box testing methodologies.

✨ Key Features
  - 🔌 Connectivity Testing: Ping tests, DNS resolution, DHCP validation, and reconnection scenarios
  - ⚡ Performance Analysis: Bandwidth measurement using iperf3, latency testing under load
  - 🛡️ Stability Validation: Long-duration tests, continuous monitoring, multiple reconnection cycles
  - 🔒 Security Assessment: Port scanning, UPnP detection, MAC filtering validation
  - 📊 Automated Reporting: Detailed test metrics collection and performance analytics
  - 🎛️ Configurable Setup: Interactive configuration for different router environments
  - 🔧 Cross-Platform Support: Works on Windows, macOS, and Linux
